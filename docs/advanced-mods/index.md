# Advanced Modding

Beyond content addition / modification, a variety of more advanced hooks have been provided to allow a mod to edit fundemental features of how Afnm works.

- [Lifecycle Hooks](lifecycle-hooks) — intercept combat, crafting, events, loot, and more
- [Adding Screens](adding-screens) — create full custom UI screens
- [External Network Requests](networking) — fetch data from remote APIs and servers
- [Translation](translation) — localise your mod content
